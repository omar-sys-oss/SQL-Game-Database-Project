# SQL Database Project — Game Data Management

##  Description

Ce projet consiste en la conception et l’implémentation d’une base de données relationnelle pour un jeu vidéo.
Il comprend la création du schéma de base de données ainsi qu’un ensemble de requêtes SQL permettant la gestion des données.



##  Contenu du projet

###  GameDatabase_Schema.sql

Contient la structure complète de la base de données :

* Création des tables
* Clés primaires
* Clés étrangères
* Contraintes d’intégrité
* Relations entre entités

---

###  Database_Requests.sql

Contient différentes requêtes SQL pour manipuler les données :

* SELECT (consultation)
* INSERT (ajout)
* UPDATE (modification)
* DELETE (suppression)



##  Technologies utilisées

* SQL
* Base de données relationnelle
* Modélisation de données



##  Objectifs du projet

* Concevoir une base de données structurée
* Assurer l’intégrité des données
* Implémeonnénter des relations entre entités
* Manipuler les des via des requêtes SQL
